스터디장: 강정훈

스터디원: 강두원,박상민, 이예진, 정훈서,김태형, 이은비, 구미진, 구민지, 김대훈, 이슬아, 김지성, 이예지, 류지예


  (1) 주2회 진행 화/금 진행 
  * 금: 알고리즘 학습(강의), 주말과제 - 3문제  

  * 화: 문제풀이, 평일과제 - 2문제  

  (2) 금요일 모임 알고리즘은 돌아가며 수업을 준비 (지원자, 지원자 없으면 뽑기)  

  (3) 과제 풀이: 스터디 전날 문제 주인 선정(지원자, 지원자 없으면 뽑기)
